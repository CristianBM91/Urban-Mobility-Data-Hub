Urban Mobility Data Hub

Un espacio de datos federado para la movilidad urbana

¿Qué es UMDH?

Urban Mobility Data Hub (UMDH) es un espacio de datos experimental orientado a la movilidad urbana inteligente, basado en tecnologías FIWARE y estándares europeos como IDSA y GAIA-X.

Arquitectura

El ecosistema integra EDC/DSC, Orion-LD, TM Forum APIs, Keycloak y APISIX para ofrecer un entorno seguro e interoperable de intercambio de datos.

Objetivos

  • Facilitar la interoperabilidad entre actores del ecosistema urbano.
  • Garantizar la soberanía y trazabilidad del dato.
  • Fomentar la innovación y la investigación en movilidad sostenible.
